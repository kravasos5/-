Можете запустить вручную или с помощью start.bat

------ИНСТРУКЦИЯ ПО ЗАУПСКУ ВРУЧНУЮ:

1)запустить csv_creator.py
	после запуска этого файла в папке должен появится файл base.csv
2)запустить count_creator.py
	после запуска этого файла в папке должен появится файл count.json

2)запустить parser_v1.py (убедитесь, что parser_v1.py, parser_module.py, csv_creator.py, count_creator.py находятся в одной папке,
			  а также в этой папке должна быть папка "data")
	
	после запуска parser_v1.py в папке появятся файлы:
		1)index.html (код сайта)
		2)all_news_dict.json (все новостные страницы на сайте, этот файл нужен для дальнейшей работы кода)
		3)AKKERMANN.json (все сайты, которые описуют инновации, один из предоставленных вариантов решения кейса)
		4)в папке "data" появятся коды каждой просмотренной новостной страницы
		5)в файл base.csv были записаны все подходящие файлы (его можно открыть в excel)
			-----ИНСТРУКЦИЯ ПО ОТКРЫТИЮ base.csv в excel:
			1)в ленте excel выбрать "данные"
			2)в окошке "получение внешних данных" выбрать "из текста"
			3)указать путь к base.csv и нажать "Импорт"
			4)указать формат данных "с разделителями"
			5)нажать "Далее"
			6)поставить флажок на запятой
			7)нажать "Далее"
			8)выбрать формат текстовый
			9)нажать "Готово"	
			10)выбрать место импорта
			11)нажать 'ок'
			если возникли трудности, смотрите подробную инструкцию на странице:
			"https://microexcel.ru/otkrytie-csv/"
